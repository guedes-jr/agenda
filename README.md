# Agenda de Evendos

## Tecnologias:
 - Django

## Instalação 
1. **Criar um Ambiente Virtual**:
   ```sh
   python -m venv myenv
   ```

2. **Ativar o Ambiente Virtual**:
   - No Windows:
     ```sh
     myenv\Scripts\activate
     ```
   - No Unix ou MacOS:
     ```sh
     source myenv/bin/activate
     ```

3. **Instalar Dependências**:
   ```sh
   pip install -r requirements.txt
   ```
